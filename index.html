Welcome to Axion Engine

Axion Engine is a powerful, open-source game development engine designed for developers of all skill levels. Built as a fork of Godot, it provides the flexibility and tools needed to create both 2D and 3D games efficiently.

Key Features

  • Cross-platform support for desktop and mobile devices.
  • User-friendly scripting interface.
  • Extensible architecture for custom modules.
  • A rich library of assets and resources.
